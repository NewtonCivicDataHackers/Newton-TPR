# Section 177 — Football game day parking regulations

Parking restrictions in effect only on Boston College football game days. Two
lists by enforcement window:
- (d) "2h before to 2h after" game commencement — ~99 entries
- (e) "7:00 a.m. to 5h after" — 3 entries

One row per atomic restriction. Most entries are "STREET, both sides[, from X
to Y / from X easterly N feet]." Commonwealth Avenue North/South Drive are
headings with numbered (1)/(2)/(3) sub-clauses → one row per sub-clause
(street = the drive, source_text street-prefixed), the bare heading not a row.
102 rows. Verified by deterministic parse; source_text unique. Manifest
counting = verified_by (heading/sub-clause mix + two lists).
